# NFL Matchup Analytics Platform

A full-stack web application for tracking and analyzing football team and player statistics with advanced analytics capabilities.

## Project Overview

This platform provides comprehensive football statistics including team performance metrics, player game stats, and defensive analytics over configurable time periods.

## Tech Stack

### Backend
- **Django** - Python web framework
- **Django REST Framework** - API development
- **PostgreSQL** (assumed) - Database

### Frontend
- **React** - UI framework
- **React Router** - Client-side routing
- **Tailwind CSS** - Styling
